# Hardware Specification

## 1. Edge Device

- **Processor**: ARM-based CPU (e.g., Raspberry Pi 4 or NVIDIA Jetson Nano)
- **Memory**: 4GB RAM or higher
- **Storage**: 32GB microSD card or larger

## 2. Camera

- **Resolution**: 1080p HD
- **Frame Rate**: 30 FPS
- **Interface**: USB or CSI

## 3. Networking

- **Connectivity**: Ethernet or Wi-Fi (802.11ac)
- **Protocols**: TCP/IP, HTTP
